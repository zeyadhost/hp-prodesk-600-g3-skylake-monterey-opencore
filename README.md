# HP ProDesk 600 G3 SFF Skylake Monterey OpenCore

OpenCore EFI for installing macOS Monterey on an HP ProDesk 600 G3 SFF with a Skylake CPU and Intel HD 530 graphics.

This EFI is intended for this hardware profile:

```text
PC: HP ProDesk 600 G3 SFF
CPU: Intel Core i5-6500, Skylake
GPU: Intel HD Graphics 530
RAM: 8 GB DDR4
Target macOS: macOS Monterey 12.7.x
Target disk: SATA SSD
Audio codec: Conexant CX20632
Ethernet: Intel I219-LM class onboard LAN
Bootloader: OpenCore 0.7.9-era config
SMBIOS target: iMac17,1
```

## Important

You must generate your own SMBIOS values before using this EFI.

Open `EFI/OC/config.plist` and replace the placeholder values with values generated by GenSMBIOS:

```text
SystemProductName: iMac17,1
SystemSerialNumber: CHANGEME_SERIAL
MLB: CHANGEME_MLB
SystemUUID: 00000000-0000-0000-0000-000000000000
ROM: set later from your Ethernet MAC address or another valid 6-byte value
```

Use GenSMBIOS to generate a fresh serial, board serial/MLB, and UUID. Do not reuse the placeholders and do not reuse someone else's serials.

## Current Notes

This EFI is adapted for Skylake HD 530 rather than Kaby Lake HD 630.

Key choices:

```text
AAPL,ig-platform-id: 00001219
Audio: alcid=23 for Conexant CX20632
XhciPortLimit: false
UnblockFsConnect: true
CtlnaAHCIPort.kext: disabled
SSDT-AWAC.aml: enabled
SSDT-RHUB.aml: disabled
IgnoreInvalidFlexRatio: true
AllowNvramReset: true
RealtekRTL8111.kext: disabled
CpuTscSync.kext: disabled
```

The current boot args are intentionally minimal while testing Recovery boot:

```text
alcid=23 npci=0x2000
```

If audio does not work after installation, try `alcid=20` or `alcid=28`.

USB mapping should be done post-install for the exact machine and ports. Do not use OpenCore Legacy Patcher unless you have a specific unsupported-hardware issue that actually requires it.

## BIOS Checklist

```text
Disable Secure Boot
Disable Fast Boot
Disable CSM / Legacy Boot
Use UEFI boot
Set SATA mode to AHCI
Disable VT-d, or adjust DisableIoMapper accordingly
Set iGPU / DVMT / video memory high enough, 128 MB or higher
```

## Installer Layout

For a Windows-created online recovery USB, the FAT32 USB root should look like:

```text
EFI/
com.apple.recovery.boot/
```

Create `com.apple.recovery.boot` with OpenCore's `macrecovery.py`.
